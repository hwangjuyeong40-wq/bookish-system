# 📦 Household Logistics Insight Dashboard

### 🔍 프로젝트 개요
한·중 생활용품 배송 및 포장 이슈를 분석하여 물류 품질 개선 인사이트를 도출하는 대시보드 프로젝트입니다.

### 🛠 기술 스택
- **Language:** Python 3.14
- **Framework:** Streamlit
- **Visualization:** Plotly
- **Data Analysis:** Pandas
- **Deployment:** Streamlit Community Cloud

### 📊 주요 기능
- **다국어 물류 이슈 비교:** 한국어와 중국어 리뷰의 감성 및 이슈 분포 시각화
- **만족도 지표 설계:** 긍정/중립/부정 가중치를 활용한 품목별 만족도 지표(KPI) 도출
- **실시간 데이터 필터링:** 품목, 언어, 감성별 리뷰 상세 조회 기능

### 🏗 데이터 파이프라인 (V2 로드맵)
[프로젝트 아키텍처 다이어그램 삽입 위치]
* 현재: CSV 로컬 데이터 활용
* 향후: SQLite DB 연동 및 쿼리 최적화 예정

### 📝 프로젝트를 통해 배운 점
1. 데이터 정제와 정량화의 중요성 (정성적 데이터를 어떻게 수치화할 것인가)
2. Streamlit을 활용한 비즈니스 대시보드 배포 경험
3. 데이터 시각화의 목적성 (의사결정을 돕는 차트 설계)
